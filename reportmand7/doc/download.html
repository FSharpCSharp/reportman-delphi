Download

To download Report Manager you must go to sourceforge project.

There, you will find the last release of the following packages:

  • Report Manager Designer application
  • Report Manager components package for Delphi /Builder/Kylix.
  • Report Manager ActiveX component, and Internet Explorer plugin.
  • Report Manager .Net library

Note: The designer packages also contains ActiveX (Windows) and Web Report Server (Windows and Linux)
