Release notes

What is new

Version 2.9 include this new features (January 2010)

  • Unicode support for some database drivers (DAO included, if the driver supports it)
  • Enhaced compatibility with Windows 7 and Vista, some video drivers worked incorrectly whe Themes were enabled.
  • Database drivers updated to the provided by Delphi 2010 compiler (DBExpress)
  • .Net library updated, dot matrix printer support and bug fixes.
  • Enhaced unicode support for PDF output.

Version 2.8 include this new features (June 2009)

  • Paral.lel unions, capability to merge two datasets into a single one sharing some columns (easy multi-dataset columnar reports)
  • New parameter type, Initial value, to allow parameter initialization when the report is read, before execution and before showing parameters.
  • Updated zeos database drivers (Windows-Linux)
  • Windows Vista compatibility (some video drivers caused problems before)
  • Updated database drivers (Windows).
  • Enhaced Dot net port. (Excel output implemented,Preview is now working in Linux because new Mono releases).
  • Added compatibility for Delphi 2007/C++Builder 2007 and Delphi 2009
  • Added compatibility for Indy version 10.
  • Bugfixes and small enhacements in the Designer.

Version 2.7e include this new features (March 2008)

  • Enhaced Dot net 1.1 and 2.0 support, preview and print working in latest linux Mono releases.
  • Simplified text search engine in preview.
  • Small features for ActiveX and GNU C library.
  • Bugfixes and performance enhacements.

Version 2.7 include this new features (September 2007)

  • Enhaced Dot net 1.1 and 2.0 support
  • Design scale (zoom in and out while designing the report)
  • Adding filters to Mybase driver using parameters.
  • Adobe Reader 8 compatibility fixes (Truetype linking and embedding)
  • Bugfixes

Version 2.6e include this new features

  • Enhaced Dot net 1.1 and 2.0 support
  • String substitution expression parameter type
  • Bugfixes

Version 2.6 include this new features

  • Dot net port, supporting Dot net 1.1 and 2.0
  • Descriptions and hints for parameters can now be translated to multiple languages in the same way report labels are.
  • Parameter validation expression: allow checking for conditions when user enter parameters (also supported on web server)
  • Shared image property, to optimize size of pdfs and metafiles.
  • Enhaced web server, language setting is available as a input parameter.
  • Bugfixes

Version 2.5d include this new features (February 2005)

  • Enhaced C# port
  • UUEncoded image support
  • In Windows, environment variables allow the load for decimal separator, date separator etc
  • Bugfixes

Version 2.5 include this new features (December 2005)

  • Preview window can now show more than one page at a time, right click on entire page icon
  • Search dataset for parameters, allowing the user to search for parameter values
  • A C# port of the engine can be tested, at data configuration select dot net driver.
  • Bugfixes (external sections and minors)
  • See http://reportman.sourceforge.net/version/dotnetport.html for more information

Version 2.4 include this new features (May 2005)

  • New parameter type, multiple selection type parameter.
  • Search dataset for parameters, allowing the user to search for parameter values
  • Parameter dbxconnections can be used to alter path to configuration file
  • Enhacements on charting
  • Experimental Unicode fonts truetype support on PDF export.
  • Report file can be stored as XML (prefered save format), experimental.
  • Bugfixes

Version 2.3 include this new features (January 2005)

  • Great performance enhacements in Linux, linked querys, subreport and others.
  • Startup performance enhacement for all command line tools.
  • Truetype font embedding speed up
  • Enhaced ActiveX control, printer selection, saving metafiles...
  • Other minor enhacements (default font in designer,optimized tiled draw, obj.inspector font selection...)
  • Partial unicode enabled designer (object inspector) for Windows NT systems, Linux already supports this.
  • Group page count element implementation.
  • PDF417 barcode printing (2d barcode allowing 1024 bytes per barcode)
  • Enhaced export to Excel (single sheet-multiple sheet)
  • Use environment variable (REPMANUSEKPRINTER=true) to integrate the print engine with KDE.
  • Remote report parameter ask and modify, from components, ActiveX and shared library API.
  • Allow different page sizes, orientations and paper bins, in a single document (Windows,PDF, experimental)
  • Report language selection at runtime (M.LANGUAGE for example to print several invoices in diferent languages in the same report)
  • New expression evaluator functions (TextHeight, ...)
  • Delphi 2005 support
  • Since 2.2 version, for client/server report comunications, the client application must be at least same version as server, older versions require same version in client and server to work.
  • Bugfixes

Version 2.2 include this new features (August 29th 2004)

  • Truetype font linking and embedding (also Type1 linking in Linux)
  • Master fields for in memory tables, to enhace query link performance
  • Performance enhacement in DBExpress linked querys
  • Paper source selection and duplex printing at page configuration
  • Added functions to exp.evaluator (chartrange, getvaluefromsql, imageop,barcodeop)
  • Other minor enhacements (default font in designer,optimized tiled draw...)
  • Important bugfixes in .Net native version
  • Bugfixes

Version 2.1 include this new features (May 14th 2004)

  • Background image in sections for easy form filling and printing.
  • Custom output, generate custom text file format output
  • E-mail report (pdf) feature from preview
  • Expanded align properties (all client alignment, top-down alignment...), this will allow flexible drawings in dinamically resized sections.
  • Child subreport support for repeatable group header sections.
  • Enhaced Teechart properties
  • New export formats: Comma Separated Values, Scalable Vector Graphics, Custom text output
  • Italian, german and catalan language support.
  • Enhaced TCP Server configuration, manage active connections.
  • New Zeos library version, stable 6.1
  • Bug fixes

Version 2.0 include this new features (December 22th 2004)

  • Native .Net support, compiled with Delphi for .Net.
  • HTML output
  • Full Report Library support (reports stored inside database tables with a tree structure)
  • Enhaced security model for web server and TCP report server
  • Enhaced ActiveX
  • Enhaced designer,data browser with drag&drop feature, categorized object inspector, documentation and more.
  • Report compiler (Windows only), generate a executable independent report (About 300Kbytes).
  • Enhaced Borland C++ Builder 6 support.
  • Some of minor features not listed here.
  • Notes: This upgrade requires updates for client and server, that is a 1.6 client does not work with a 2.0 server and a 2.0 client does not work with 1.6 server.

Version 1.6 include this new features (August 7th 2003)

  • Multiprocessor support (smp).
  • New output driver, text only with printer escape codes to print lighting fast in any dot matrix printer, in Windows and Linux, also to save the report result as plain text.
  • New shared libraries libreportmanapi.so and libreportmanapiqt.so to access the engine with GNU C and any language capable of calling dynamic link library functions (Reportman.ocx for Windows)
  • Zeos library drivers support
  • Excel export support for Microsoft Windows
  • Allow report definition streaming in three formats binary-zlib, binary and text , so you can track changes in your reports with cvs
  • Plain text file input support as a database file with visual configuration dialog
  • New method for VCL components and ActiveX, executeremote to execute reports stored in report servers, also available in ActiveX and GNU C interface.
  • New property editors for Delphi 5 (TRpAlias)
  • Enhaced command line tools (-preview -pdialog options added), all tools can work now with standard input and standard ouput.
  • ActiveX composite reports feature.
  • New methods for drawing text and graphic elements usable from the expression evaluator.
  • Enhaced designer - New database connections/datasets configuration dialog, active help in properties in the object inspector...
  • Enhaced support for Borland C++ Builder (added CLX package and more documentation)
  • Enhaced documentation
  • Portuguesse language support thanks to Warlei Alves
  • Bug fixes
  • Report Manager is now a Project Jedi Alliance member
  • Notes: This upgrade requires updates for client and server, that is a 1.5 client does not work with a 1.6 server and a 1.6 client does not work with 1.5 server. This is also true for ActiveX Internet explorer plugin. Replace dbxdrivers old file with the new one for zeos support.
